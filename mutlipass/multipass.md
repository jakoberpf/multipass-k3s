# Multipass k3s cluster
(https://jyeee.medium.com/kubernetes-on-your-macos-laptop-with-multipass-k3s-and-rancher-2-4-6e9cbf013f58)

## Prerequisites
Install multipass via brew.
```
brew cask install multipass
```
Check version.
```
multipass version
```
## Deploy
```
./deploy.sh
```
## Destroy
```
./destroy.sh
```
